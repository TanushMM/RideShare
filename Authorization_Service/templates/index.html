API Documentation

Welcome to the API documentation for this application. Below are the details of the available routes and how to use them.

1. / (GET)

Base endpoint for the application. It renders the index.html template.

Method: GET

Response: Renders the index.html template.

2. /getJWT (POST)

Generates a JWT token based on the provided JWT from the login service.

Method: POST

Request Body: JSON containing the JWT from the login service. 

    {
        "jwt": "token_from_login_service"
    }

Response: JSON containing a success message and the generated JWT token. 

    {
        "message": "success",
        "total_server_access_token": "generated_jwt_token"
    }

Error Response: JSON containing an error message if an exception occurs. 

    {
        "message": "Error message"
    }
