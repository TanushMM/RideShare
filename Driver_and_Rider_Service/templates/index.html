API Documentation

Welcome to the API documentation. This API is designed to handle requests for both drivers and riders. The primary endpoints provided by this API are accessible under the /driver and /rider prefixes.

Authorization

This API uses JSON Web Tokens (JWT) for authentication. Every request to the /driver and /rider endpoints must include a valid JWT in the header.

To obtain a JWT, you can use the /getJWT endpoint with a POST request. The JWT should be included in the Authorization header as a Bearer token: 

        Authorization: Bearer <your-token-here>

Endpoints

  • /driver: Access driver-related functionalities.
  • /rider: Access rider-related functionalities.
